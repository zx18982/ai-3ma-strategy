# 双均线交叉策略回测看板 · AI-3MA-Strategy

> 量化工作坊 Task03 —— 用均线交叉（金叉买入 / 死叉卖出）捕捉市场趋势，并基于历史日线数据进行模拟回测与可视化分析。

## 简介

本工具是一个**纯前端、零部署**的双均线（Moving Average）交叉策略回测看板。它加载本地已存储的前复权日线数据，自动计算短/长周期均线、识别金叉/死叉交易信号，并进行模拟交易回测，最终以交互图表 + 量化指标 + 策略分析的形式呈现结果。

覆盖三只典型标的（A 股 / H 股）：

| 标的 | A 股代码 | H 股代码 |
|------|----------|----------|
| 中芯国际 | sh688981 | hk00981 |
| 比亚迪   | sz002594 | hk01211 |
| 长江电力 | sh600900 | —        |

## 功能特性

1. **数据加载**：内置 5 只股票的前复权日线数据（已内嵌，无需联网）。
2. **均线计算**：可调短 / 长 SMA 周期（默认 5 / 15），自动约束长 > 短。
3. **信号生成**：金叉（短上穿长）→ 买入；死叉（短下穿长）→ 卖出；信号交替约束。
4. **可视化**：收盘价 + 双均线 + 红▲买入 / 绿▼卖出标记；净值曲线、回撤曲线；鼠标十字光标 tooltip（遵循**涨红跌绿**配色）。
5. **模拟回测**：全仓买卖模拟，支持手续费（万三）/ 滑点（万一）开关。
6. **量化指标**：年化收益率、夏普比率、最大回撤（MDD）、累计回报、胜率、超额收益、年化波动率、成本侵蚀等。
7. **时间窗口**：可选起始 / 结束日期，限定任意回测子区间。
8. **策略分析**：数据诊断 + 策略评价 + 改进建议（动态生成）。
9. **导出**：交易记录与指标可导出 CSV / JSON。

## 快速开始

### 方式一：直接打开（推荐）
双击 `index.html` 即可在浏览器中运行。数据已内嵌，无需服务器、无需联网。

### 方式二：本地静态服务器
```bash
cd ai-3ma-strategy
python3 -m http.server 8137
# 浏览器访问 http://127.0.0.1:8137/
```

### 使用步骤
1. 左侧选择标的、设定时间窗口与均线参数。
2. 调整参数后，按钮会变为高亮「运行回测 ▶」。
3. 点击 **运行回测** 重新计算并刷新图表与指标。
4. 右侧查看净值曲线、回撤、买卖点、交易记录与策略分析。

## 技术架构

| 模块 | 文件 | 说明 |
|------|------|------|
| 页面骨架 | `index.html` | 左侧边栏 + 右侧看板布局 |
| 样式 | `css/style.css` | 涨红跌绿配色、看板样式 |
| 策略引擎 | `js/strategy.js` | SMA / 金叉死叉检测 / 全仓回测 / 指标计算（纯函数） |
| 图表 | `js/charts.js` | 纯 SVG 图表 + 十字光标交互（无第三方依赖） |
| 分析 | `js/analysis.js` | 数据诊断 / 策略评价 / 改进建议 |
| 主控制器 | `js/app.js` | 串联加载→切片→均线→信号→回测→绘图→诊断 |
| 内嵌数据 | `data/stock_data.js` | 由 `scripts/raw_data/*.json` 生成，避免 `file://` 下 fetch 失败 |

> 设计决策：原计划使用 ECharts，但 `file://` 协议下 CDN 受限，故改用**纯 SVG 自绘 + 内嵌数据**，实现双击即用的零部署目标。

## 重新生成数据

如需用最新行情刷新数据：

```bash
cd scripts
python3 fetch_data.py     # 拉取最新前复权日线到 raw_data/
# 再将 raw_data/*.json 转为 data/stock_data.js（见 fetch_data.py 内说明）
```

## 风险提示

本工具所有结论均基于**历史数据**的模拟回测，仅用于量化交易学习与研究，**不构成任何投资建议**。过往业绩不代表未来表现。
